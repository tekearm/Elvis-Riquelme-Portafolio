Proyectos destacados

Proyecto 1 — Control de Combustible

Duración: 3 meses

Descripción: Software de control de compras y entregas de combustibles a la flota de camiones de Transportes san pablo.

Objetivos:

  • Implementar una solucion para el control de consumos y analisis de estos.
  • Generar informes de analisis de comras y rendimientos.

Tecnologías: Visual Basic .net, SQL Server.

Reflexión: Aprendí a diseñar un mejor control para llevar los rendimientos y el calculo de consumos dentro de las distintas sucursales a lo largo del pais.

Proyecto 2 — Sistema de Control de Bodega

Duración: 2 meses

Descripción: Software para el control de entradas y salidas de las bodegas de repuestos.

Objetivos:

  • Generar ordenes de ingreso a taller, establecer tipo de mantencion y repuestos ocupados.
  • Stock automatico, con stock critico y aviso por email.

Tecnologías: Visual Basic .net, SQL Server, Gps Chile.

Reflexión: Mejoré mi comprensión de la arquitectura cliente-servidor.

Proyecto 3 — Modelado Predictivo

Duración: 1.5 meses

Descripción: Modelo de clasificación para predecir churn de clientes basado en señales de uso.

Objetivos:

  • Explorar datos, seleccionar features y validar modelos.
  • Crear un pipeline reproducible para entrenamiento y evaluación.

Tecnologías: Python, scikit-learn, pandas, MLflow, Docker.

Reflexión: Entendí mejor la importancia de la validación temporal y cómo asegurar que un modelo generalice fuera del set de entrenamiento.
